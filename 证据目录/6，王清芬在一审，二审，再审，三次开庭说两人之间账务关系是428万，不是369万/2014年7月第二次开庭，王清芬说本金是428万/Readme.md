法官亲自问两个人

“原、被告各自认可的借款本金是多少？”

原告：“428万元。”

二被告：“369万元。”

这段话，加上王清芬2014年2月提交给法院的诉讼请求，清楚地表达了王清芬坚定想要以428万为本金计算两人账务的意图
