3，这个证据是王清芬在再审现场，对上一份王清芬和林国付签字的证据进行质证时，说的一句话：“谁都知道是我去和林国付谈判的啊”。

这是一句东北话，在当时的语境里的意思就是：所有人都知道，是我去和林国付谈判的。

这里有一个插曲值得说一下。

这张证据原件上原本没有日期，王清芳在即将上法院之前不久，给自己后补了一个日期，而且她凭借记忆写，还把日期写错了。你说王清芳为啥就这么直接在原件上随便乱写？

王清芳就有这种习惯，包括我上传的很多图片上的比如判决书上，调查笔录上，王清芬让王清泉送来的对账清单上，很多我拍照的内容，你都能看到王清芳的批注。这是律师调卷之后，王清芳自己写在卷宗上的。

她就喜欢这么干，她甚至认为她的批注还能起到画龙点睛的作用。我要跟她争吵，她就说人家王清芬在借据上都画来画去也没人管，想咋整咋整，想咋说就咋说，我却把自己把自己束缚起来了。

她说的也有道理，比如王清芬说她没在631号调解书上签字，本来我只要说她撒谎就可以了，但是有追求严密的人，就得问，你咋知道王清芬是撒谎，你咋知道不是她忘了呢？

所以我就尽力把王清芬撒谎的整个过程都描述出来，包括刚签完第一个月，她就说：你妈签字我在旁边同意的，字不是我签的。这不是一个“忘了”的人，应该“想起来”的过程。

然后让读者自己去得出结论，王清芬在撒谎。但是如果我不直接说王清芬撒谎，有些人真就看不出来这个简单的结论。

最后还可能是由于我说得太严谨，把自己束缚住了。这种风格贯穿整个文档。

我张晓东对这件事付出的心血，我每一个字，每一句话，都力求不要让任何刁钻的人放到放大镜下找出任何道德瑕疵。我是在用圣人的标准要求自己，这也是我的力量源泉，但是她王清芳就随便在原件上写东西。

调查笔录，判决书这种原件就罢了，连找到一个王清芬和林国付签订的原始协议上，都去写东西，这对我造成极大的困扰，给我带来极大痛苦。这件事和王清芳王清慧骂我，削弱了我99%以上的力量。

这要是一个别人的事，或者别的公司的事，那你爱几把咋滴咋滴，爱几把倒闭就倒闭。最后我也只能想开点，这不是我的事，爱几把咋滴咋滴吧。

好像王清芳还没在借据上写东西，还有那么一点数，不能乱写东西的证据，现在都已经过了质证阶段。

从卷宗里调出来之后又写上东西的，也很容易分辨是后写上去的，想要没写东西的，也可以从卷宗里复印出来的。

我对王清芬的仇恨，远远超出你们的想象。远远远远，远超出你们任何一个人的想象。

王清芬不仅是一个撒谎的人那么简单，也不仅是一个几十万的诈骗犯那么简单，也不仅是一个想把我父母置于死地的仇人，更是让我母子关系决裂，让我全家的不幸，都指数级上升的人。

我还曾经听说王清芬还说过对我怎么怎么好？哎？有傻逼会相信么？没轮到你头上你当然不知道该不该相信，所以就拿你举例，当我想整死你妈和你爸的时候，我说我对你怎么怎么好，你会相信么？

如果这个人是你大姨，你大姨想诈骗你妈你爸，然后跟你说，这是大人的事，小孩别管，你小时候我对你多好呀，你可不能管呀。这是人话么？

爸妈和大姨，而且是一个诈骗嫌疑人的大姨，而且还是诈骗你爸妈的大姨，谁远谁近，这还是一个需要考虑的问题么？

如果你都不相信这种“我小时对你多好”的鬼话，你为什么要觉得，我就能相信呢？因为我长得像傻逼？

以后我不管做出怎么对待王清芬和其家人的事情，只要是法律范围内的，我请你们都要记住我今天的话。

2016年11月2日，再审当天，这个王清芬和林国付签的原件，我提交给法官时，我在法庭当庭上当庭就说了，大概意思就是：“这个右下角的日期是王清芳后写上去的，日期还写错了，不是这个证据原件上的内容，其他都是王清芬和林国付签的内容”。

王清芬大概听到了我说这些话，质证时，王清芬先说：“字是我签的，但是没有签在这个文件上，是他们把我的签名，贴到这个文件上去了”

啊，你看看这个人啊，是一个什么样的内心世界啊。但是马上又是恍然大悟似的，“啊，好像是有一个文件”，

然后法官问她，是否鉴定，王清芬说：“我不鉴定”，“谁都知道是我去跟林国付谈判的呀，这谁都知道呀”。

她说的“谁都知道是我去和林国付谈判的啊”，这句话我觉得比这个证据都关键，电话录音，加上这句庭审时对上面提到的这个证据的质证意见，就可以证明是王清芬主导的和林国付的谈判，并放弃了现金流动性，要出了一堆房子。

但是王清芬说“谁都知道是我去和林国付谈判的啊”，这句话，没有写入再审的庭审笔录，整个过程需要有权力调取再审现场录音录像才能听到。

包括我当庭说，这个日期是后补的，都可以在庭审录音录像中听到。

既然我想起来这段，我就写上，给人类历史上将来可能会听到这段录音的人一个解释。

但是卷宗里的复印件是没有右下角标记的日期的，应该是复印的时候就没印日期。

这整件事给我造成的困扰外人难以想象。好好的一份文件，王清芳就给随便写上东西，然后复印的时候，为了恢复原有的样子，再不去印后加的这个东西。

为什么一件原本光明正大的事情，非要搞得这么凭空添加了这么多痛苦和可能被人指指点点的事。这让我非常痛苦。

客观上看，不是一个很重要的证据，但是离我心目中的圣人标准相差甚远，就足以给我造成痛苦。这些痛苦跟王清芳把我当成狗一样骂几乎不相上下。

但是我认为这和王清芬把一个借据上的2分利改成3分利，拿去起诉，性质还是不一样的。

王清芬是虚构和隐瞒事实，王清芳看了2改3的借据之后，质证意见是：不是3分，是2分。

王清芳并没有虚构和隐瞒事实的目的，王清芬看了右下角增加了一个日期的协议后，质证意见先是：所有人都知道是我和林国付去谈判的。

这个插曲，只有找到当庭的录音录像才能听到。我写在这里，是留给后人能查看到当时录音录像的人一个解释，避免形成“历史疑团”，张晓东在庭审录音中说的这个“日期写错了”，是什么意思，是不是有造假嫌疑？

这个证据能让王清芬说出那句“谁都知道是我去和林国付谈判的呀”，这句话，可能还更重要。这份王清芬和林国付签订的材料，我们就假设不采纳。

但王清芬说的这句话：“谁都知道是我和林国付去谈判的”，这也是一个原被告双方都认可，而且和现有法律文件，和现有证据，也都完全不冲突的陈述。应该采纳。
