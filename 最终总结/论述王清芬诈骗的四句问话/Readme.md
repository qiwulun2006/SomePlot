这实际上是一个常识：按利滚利比不按利滚利能要出更多的钱。

王清芬自己也分别计算了369万，按利滚利应该多少钱，不利滚利应该多少钱，究竟是哪个多哪个少，就是王清芬让王清泉送来的算账明细。

我们现在更知道了“因为起诉要有证据，所以我就只能用428万起诉了”这是一句谎言，所以你现在告诉我：

王清芬为什么要用428万起诉，不用369万起诉？

第一，不是因为起诉要有证据所以只能用428万。王清芬也当庭主动表示，必须按428万算，不能按369万算。

第二，王清芬清楚地知道，428万比369万能要出更多钱，至少，王清芬在看完了王清芳写给林国付的信之后，她知道了用428万能比369万要出更多钱。

第三，王清芬亲自计算过369万能算出多少钱（王清泉送来的账目），但是在起诉时，放弃了这种算法，给出了一套按428万算的诉讼请求。

我记得有个知名的刑法教授叫罗翔，曾经说过，判案要按照普通老百姓的观点来进行审判。不能按照那些哲学家的观点来判案。

现在，王清芬费劲心思保存了220万的借据，还虚假诉讼，用两人已经宣布作废的220万来起诉，还告诉张晓东：之前看你妈是我妹妹，我没“重复算”，现在我跟之前要的不一样了，现在我要“重复算”

而且，王清芬还看了王清芳写给林国付的信，虽然信中没有220万的事，但是王清芬还在法庭上当庭分享了读信心得：用220万算能算出来更多钱。

现在，你来告诉我，王清芬用220万来起诉总价值428万，是想要和369万一样的钱，还是想要比369万更少的钱，还是想要比369万更多的钱？

如果你给出一个结论：王清芬用428万起诉，是为了让自己要更少的钱，是为了让王清芳分到更多的钱。我觉得你的脑袋已经不是不是那种普通老百姓的脑袋，这已经是一个应该去治一治的脑袋了。

你要能给出这么一个结论来，你的良心不会痛么？菜市场大妈的心不会痛么？
